Crear y activar entorno virtual:
1. Instalar virtualenv:
   pip install virtualenv

2. Crear entorno virtual:
   virtualenv venv

3. Activar entorno virtual:
   En PowerShell:
   .\venv\Scripts\Activate
   En cmd:
   venv\Scripts\activate

4. Verificar entorno virtual:
   python --version
